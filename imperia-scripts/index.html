Alarma

Alarma que se activa con ataques personales superiores a 999 soldados, re-diseñada para V6.

Características

  • Sonido
  • Auto-recarga para evitar retrasos con el servidor
  • Notificaciones de activación de la Alarma (Se requiere previa autorización)

Requisitos

Se requiere tener instalado Tampermonkey.

  • Si usas Chrome
  • Si usas Firefox

Instalación de Alarma

Click para Instalar

Guía de uso

A tener en cuenta; después de la instalación no es necesario activar la alarma manualmente, cada vez que se inicie sesión la alarma se activará. Para garantizar el correcto funcionamiento se recomienta no tener más pestañas junto con el juego, ya que los navegadores pueden interrumpir el funcionamiento del script.
